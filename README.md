# stream-ui

Flexible UI framework that streams UI to the user, guided by an AI agent.

Part of a planned **agent-tools** framework family.

## Install

```bash
bun add stream-ui
```

## Usage

```ts
import { VERSION } from 'stream-ui'

console.log(VERSION)
```

## Develop

```bash
bun install
bun run dev          # watch + rebuild
bun test             # run tests
bun run lint         # biome check
bun run typecheck    # tsc --noEmit
bun run build        # produce dist/
```
